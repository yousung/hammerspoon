- modules/inputsource_aurora.lua
  - 한영 오로라 설정
  - 키보드 한영에 따라서 헷갈리지 않도록 상단바의 컬러를 넣어줌

  - [참고](https://www.hammerspoon.org/docs/hs.screen.watcher.html)
